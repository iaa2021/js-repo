3️⃣ JavaScript Functionality
const socket = io();
const form = document.getElementById('form');
const input = document.getElementById('input');
const messages = document.getElementById('messages');

const socket = io(); → Connects to a Socket.IO server (real-time messaging).

form, input, messages → Grab the HTML elements by ID so you can interact with them in JS.

Handling Form Submission
form.addEventListener('submit', (e) => {
    e.preventDefault(); // Prevent the page from reloading
    const message = input.value.trim(); // Get text and remove extra spaces
    if (message) {
        socket.emit('chat message', message); // Send message to server
        input.value = ''; // Clear input field
    }
});

Prevents the default form behavior (which reloads the page).

Gets the user’s message and sends it to the server using Socket.IO.

Clears the input so the user can type the next message.

Listening for Incoming Messages
socket.on('chat message', (msg) => {
    const item = document.createElement('li'); // Create a list item
    item.textContent = msg; // Set message text
    messages.appendChild(item); // Add it to the messages list
    messages.scrollTop = messages.scrollHeight; // Auto-scroll to the newest message
});

Every time the server broadcasts a message, this listener runs.

Creates a new <li> element and adds it to the chat.

Scrolls to the bottom so the latest message is visible.

4️⃣ How This Works End-to-End

User types a message and clicks Send.

JS captures the message and emits a chat message event to the server.

The server receives the message and broadcasts it to all clients.

All connected clients receive the event and append the message to the chat list.

Messages appear in real-time without page reloads.
